# HIT Shooting Game - User Manual

Welcome to **HIT Shooting Game**! This guide will help you understand the controls and navigate the game's settings.

---

# Gameplay Controls

| Action | Key |
|--------|-----|
| Move | **W, A, S, D** |
| Sprint | **Left Shift** |
| Slide | **Left Alt** |
| Grapple / Hook to Walls | **R** |

## Movement

- **W** – Move Forward
- **A** – Move Left
- **S** – Move Backward
- **D** – Move Right

## Sprinting

Hold **Left Shift** while moving to sprint.

## Sliding

Press **Left Alt** while sprinting or moving to perform a slide.

## Grappling Hook

Press **R** to use the grappling hook. This allows you to hook onto walls and quickly move toward them.

---

# Settings Menu

Use the **Arrow Keys** on your keyboard to navigate through the settings menu.

## Volume

- **Left Arrow** – Decrease volume
- **Right Arrow** – Increase volume

## Resolution

- **Left Arrow** – Select the previous resolution
- **Right Arrow** – Select the next resolution

## Background Color

The background color uses the **RGB (Red, Green, Blue)** color model.

### Selecting a Color Channel

Use the **Left** and **Right Arrow** keys to choose which color channel you want to edit:

- **R** – Red
- **G** – Green
- **B** – Blue

### Changing a Value

1. Select **R**, **G**, or **B**.
2. Press **Enter**.
3. Type a number between **0** and **255**.
4. Press **Enter** again to confirm the value.

> **Note:** Each RGB value must be between **0** and **255**.
>
> Example:
> - R = 255
> - G = 128
> - B = 64

This would produce an orange-colored background.

---

# Tips

- Use **Sprint** and **Slide** together for faster movement.
- Use the **Grappling Hook (R)** to quickly reach walls and navigate the map.
- Experiment with different RGB values to personalize the background color.

---

Enjoy playing **HIT Shooting Game**!
